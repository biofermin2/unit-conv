* Unit-Conv 
unit conversion program


** Installation
1.install from github repository.

#+BEGIN_SRC sh
$ ros install biofermin2/unit-conv
#+END_SRC

2.load library by quickload.
#+BEGIN_SRC lisp
(ql:quickload '(:unit-conv) :silent t)

#+END_SRC

then you can use this.
that's all.
